Decepticons
🔗 Live Demo: https://hanna-006.github.io/decepticons/
Overview

Decepticons is a front-end web application that explores unconventional user interface behavior through interactive and dynamic design elements. The project focuses on creating an engaging and unpredictable user experience by leveraging event-driven programming and creative visual effects.
It serves as both a demonstration of modern web development techniques and an experimental platform for testing non-traditional interaction patterns.

Key Features

Interactive UI Components – Responds dynamically to user actions such as clicks and cursor movement

Event-Driven Logic – Built using JavaScript to handle real-time interaction flows

Custom Animations – Enhances user engagement through CSS-based visual effects

Lightweight Architecture – Runs entirely in the browser with no external dependencies

Cross-Browser Compatibility – Designed to function across modern web browsers
Technology Stack

HTML5 – Semantic structure and layout

CSS3 – Styling, transitions, and animations

JavaScript (ES6+) – Core functionality and interactivity

Getting Started

Prerequisites

A modern web browser such as Chrome, Edge, or Firefox.

Installation
Clone the repository:
Bash
git clone https://github.com/hanna-006/decepticons.git
Navigate to the project directory:
Bash
cd decepticons
Open index.html in your preferred browser.

Usage

Simply open the application in a browser and interact with the interface to experience the dynamic behaviors implemented in the project.
